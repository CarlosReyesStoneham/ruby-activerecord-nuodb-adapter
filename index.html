Welcome to the Ruby ActiveRecord NuoDB Adapter Project

This site contains details on how to install and use the Ruby ActiveRecord NuoDB Adapter. To install the driver simply run the following command for pre-release builds:

$ gem install activerecord-nuodb-adapter --pre

For release builds it's simply:

$ gem install activerecord-nuodb-adapter
